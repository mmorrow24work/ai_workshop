# AI Workshop — Zero to Hero Guide

Welcome! This repository is a practical, no-jargon guide for anyone who wants to understand and use AI tools — whether you're an IT professional, a developer, or a curious newcomer.

---

## Who Is This For?

- People who've heard about AI tools but don't know where to start
- IT professionals wanting to understand the landscape before using AI with client data
- Developers looking to go beyond the ChatGPT web interface and build with AI
